# PHP Login System

This is a simple PHP-based login and registration system with a dashboard and user posts.

## Features
- User registration
- User login/logout
- Dashboard showing user info
- Ability to post messages
- Basic CSS styling

## Installation
1. Install a local PHP server like [XAMPP](https://www.apachefriends.org/index.html) or WAMP.
2. Place the project files in the `htdocs` (or equivalent) folder.
3. Create a MySQL database named `social_network`.
4. Import the required tables (users, posts) using the provided SQL or manually create them.
5. Update `connection.php` with your database credentials.

## Usage
- Open `http://localhost/Login-System/Login%20System.php` in your browser.
- Register a new user or log in with existing credentials.
- Access the dashboard and post messages.
